# Поиск тестов по айди

import useBaseUrl from '@docusaurus/useBaseUrl';
import { DiscordMention, DiscordMessage, DiscordMessages } from '@danktuary/react-discord-message';
import DiPeekResponse from '@site/src/components/DiPeekResponse';

:::info Пререквизиты
Необходимо, чтобы у вас уже был назначен (выставлен) тест в DiSpace.  
Если его у вас нет, то [поищите по названию](./searching-tests-by-name).
:::

### Находим айди теста {#finding-test-id}

Айди (ID, идентификатор) теста можно узнать, наведя мышкой на ссылку с тестом.

<img src={useBaseUrl("/img/docs/finding_test_id.png")}/>

В браузере, снизу слева (или справа) появится текст ссылки.  
**Последнее число в этой ссылке - айди теста.**

<img src={useBaseUrl("/img/docs/finding_test_id_2.png")}/>

В примере выше, айди теста равен `14444`.

### Ищем тест в базе данных {#searching-tests-by-id}

Заходим на [Discord сервер DiPeek'а](https://discord.gg/tphsh9vsty), и вводим `/test <ID> txt` с полученным айди теста:

<DiscordMessages>
  <DiscordMessage>
    {"/test 73754 txt"}
  </DiscordMessage>
  <DiPeekResponse>

```
Зарегистрированных попыток: 266.
Извлечено с помощью DiPeek: https://discord.gg/tphsh9vsty
============================================================================================================================
||||| ТЕСТ "Лабораторная работа №14 &quot;Определение удельного заряда электрона методом магнетрона&quot;" (ID: 73754) |||||
============================================================================================================================

............................................................................................................................
```

  </DiPeekResponse>
</DiscordMessages>

:::note Заметка
Планируется больше форматов вывода: как минимум Markdown и HTML, поэтому в конце надо писать `txt`.
:::

### Не удалось найти тест? {#test-not-found}

<div style={{maxWidth:'400px'}}>
  <DiscordMessages>
    <DiPeekResponse>
      {"Не удалось найти тест с ID 12345..."}
    </DiPeekResponse>
  </DiscordMessages>
</div>

Возможно тест был *переайдирован*. В таком случае [ищите по названию](./searching-tests-by-name).

:::caution Предупреждение
Если найдёте тесты по названию, то не будьте уверены, что всё будет точно таким же.  
Преподаватель вполне мог загрузить совершенно другой тест, с другими вопросами, но с тем же названием.
:::

### Удалось найти тест, но там мало попыток? {#not-enough-attempts}

<DiscordMessages>
  <DiPeekResponse>

```
Зарегистрированных попыток: 3.
!!! ПРЕДУПРЕЖДЕНИЕ: Попыток мало, и некоторые вопросы могли быть не раскрыты.
!!! Это также применимо и ко всем другим тестам, но тут будьте особенно осторожны.

Извлечено с помощью DiPeek: https://discord.gg/tphsh9vsty
==========================================
```

  </DiPeekResponse>
</DiscordMessages>

Ну, думаю, по предупреждению выше всё понятно.  
Можете поискать другие похожие тесты [по названию](./searching-tests-by-name).

:::caution Предупреждение
Даже если в другом тесте куча попыток, не надейтесь, что вопросы в вашем тесте будут точно такие же.  
Преподаватель вполне мог загрузить совершенно другой тест, с другими вопросами, но с тем же названием.
:::

:::note Что? А почему?
Так как DiPeek собирает ответы только с _прохождений_ теста, то, если мало кто проходил тест, будет известно мало вопросов. Кто знает, может в тесте 100 вопросов, а из 5 попыток стали известны лишь 72 из них.
:::
