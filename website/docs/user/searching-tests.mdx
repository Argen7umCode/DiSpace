### Находим айди теста {#finding-id}

Айди теста можно узнать, наведя мышкой на ссылку с тестом.

<img src={useBaseUrl("/img/docs/finding_test_id.png")}/>

В браузере, снизу слева (или справа) появится текст ссылки.  
**Последнее число в этой ссылке - айди теста.**

<img src={useBaseUrl("/img/docs/finding_test_id_2.png")}/>
<br/><br/>

:::info Примечание:
Если у вас тест не открыт, или нет возможности увидеть его айди, попробуйте [найти тест по его названию](#search-by-name).
:::

### Ищем тест по айди {#search-by-id}

Теперь вводим `/test <id> txt` с полученным айди теста.

### Ищем тест по названию {#search-by-name}

Название теста, я думаю, вы должны знать перед его прохождением.  
Введите `/test search <term>` с запросом для поиска.

:::tip Совет:
В качестве запроса используйте корень одного из ключевых слов.

Если вы введёте "теология", то бот, естественно, **не выведет** тесты, которые содержат "теологии" или "теологические". DiPeek не такой умный, чтобы различать склонения, падежи и прочие сложные штуки. Лучше вводите "теолог".
:::
